# MicroPython esp8266 8x8 led matrix

```
Bekötések:

8x8     esp8266
--------------
CLK --> D4 - GPIO: 2  
CS  --> D3 - GPIO: 0
DIN --> D2 - GPIO: 4
GND --> GND
VCC --> 5V

A 8x8-as mátrixra rajzolni ezen a honlapon lehet:
http://dotmatrixtool.com/#
Itt be kell állítani a szélességet és a magasságot 8-ra, majd a generate gombra kattintva kapunk egy tömböt.
Ennek a tömbnek a tartalmát kell majd kimásolni és a mi kódunkban használni.


Referencia: https://www.instructables.com/id/Arduino-8x8-LED-Matrix/
```
